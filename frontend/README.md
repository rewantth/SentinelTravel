# SentinelTravel Frontend

React + Tailwind frontend for the SentinelTravel SOC interface.

## Prerequisite

Make sure the backend is running first at port `8000`.

```bash
uvicorn backend.main:app --reload --port 8000
```

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

Open the Vite URL shown in the terminal, usually `http://localhost:5173`.
